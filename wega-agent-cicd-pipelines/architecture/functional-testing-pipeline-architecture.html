Stage Selection (Conditional Execution)

1

Run_All_Agents_Sequential

Executes all tests sequentially using run_test.py

selectedAgent == "all"
2

Run_Agents_Parallel_Matrix

Executes all agents in parallel (max 5 concurrent)

selectedAgent == "parallel"
3

Run_Specific_Agent

Executes tests for a single selected agent

selectedAgent != "all" && != "parallel"

Execution Steps

1

Clone Repository

Clone the functional test repository from Git Harness

Repo: wega-functional-testcases
Directory: functional-tests/
Branch: <pipeline.variables.branch>
2

Create Config File

Generate config.json with environment-specific URLs and credentials

File: data/config.json
Secrets: wega_test_email, wega_test_password
3

Install Dependencies

Install Python packages and Playwright browsers

Packages: pytest-playwright, pytest-html, pytest-xdist, pytest-order
Browser: playwright install --with-deps chromium
4

Run Tests

Execute Playwright tests with retry logic on failure

Output: reports/*.html, screenshots/*, test-results*.xml
Retry: On failure, retry once
Failure Strategy: MarkAsSuccess (continue pipeline)
5

Upload Artifacts to GCS

Upload HTML reports and screenshots to Google Cloud Storage

Bucket: wega-test-artifacts
Path: functional-tests/<executionId>/<agent>/reports|screenshots
