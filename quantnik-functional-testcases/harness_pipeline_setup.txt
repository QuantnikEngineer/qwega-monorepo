# Harness Pipeline Setup Instructions

1. Ensure Python (3.8+) is installed in your pipeline environment.

2. Install dependencies:
   pip install -r harness_requirements.txt

3. Install Playwright browsers:
   python -m playwright install

4. Run your test suite:
   python run_test.py

5. Collect the generated HTML report (test_suite_report_*.html) as a pipeline artifact.

# Minimal harness_requirements.txt:
pytest==9.0.3
playwright==1.59.0
pytest-html==4.2.0
